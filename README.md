## General info
This project is simple landing page made during CodersCamp 2020.
	
## Technologies
Project is created with:
* HTML
* CSS
* SCSS

## Requirements
- Box-model
- Cascading CSS
- CSS selectors
- Semantic HTML
- Linking CSS to HTML
- Styling text
- External fonts/icons
- Flexbox and/or CSS Grid
- Position (absolute, relative)
- Animations - keyframes
- Simple contact form
- Responsive Web Design
